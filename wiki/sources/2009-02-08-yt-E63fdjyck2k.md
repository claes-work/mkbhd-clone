# HD Tutorial: Customize Command Prompt (@mkbhd, 2009-02-08)

## Summary
Origin-era customization screencast. Marques (age 15) shows how to restyle the Windows
command prompt via right-click the CMD icon → Defaults: font (he picks Lucida Console
size 14), layout (pixel width/height for a widescreen console), and colors — where he
sets his text to a bright **red** on a black background for "very sharp contrast," plus a
purple/white popup scheme. He notes he applies the same red-and-black look to WebKit so
there's "a little something to look at." An early, explicit statement of personal visual
taste — red on black — years before the crimson-red MKBHD studio brand.

## Key claims
