# HD Tutorial: Safari and Webkit Addons (@mkbhd, 2009-02-04)

## Summary
Origin-era browser how-to closing out his Feb-2009 browser cluster (IE, Firefox, Safari).
Marques shows how to find and install **plugins for Apple Safari 3.2 on Windows** — and the
same plugins for **WebKit** nightly builds — via apple.com → Downloads → Safari →
"plugins for Safari on Windows." He lists the six available plugins (Adobe Reader, Flash,
QuickTime, Java, RealPlayer, Windows Media) and shows how to check which are already
installed via Help → Installed Plug-ins. He concedes Safari's plugin experience isn't as
rich as Firefox's add-ons but still recommends it. He ends by **inviting viewers to send
video/tutorial requests** — an early audience-engagement tell.
