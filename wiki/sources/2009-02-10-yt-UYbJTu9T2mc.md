# HD Tutorial: Compatibility Mode (@mkbhd, 2009-02-10)

## Summary
Origin-era troubleshooting how-to. Marques (age 15) explains how to run an older program on
**Windows Vista/XP** via **Compatibility Mode**: put the program's icon on the desktop,
right-click → Properties → Compatibility tab, check the box, and pick a target OS (e.g.
Windows 95) plus options like running in a 640×480 window, disabling visual themes for
speed, or running as administrator. He honestly notes that some programs (Notepad, the
demoed Word) have the option **grayed out** and won't run in compatibility mode — and cautions
the program may look or behave slightly differently afterward.

## Key claims
- [2009-02-10] **How-to:** run a legacy app via the **Compatibility** tab of its shortcut's
  Properties; select an older OS and optional flags (640×480 window, disable visual themes,
  run as admin).
- [2009-02-10] Not all programs support it — the checkbox is **grayed out** for some — and
  compatibility mode may change a program's look/behavior slightly.
