# Siri vs Google Voice Search! (@mkbhd, 2012-07-01)

## Summary
A side-by-side voice-assistant head-to-head — Siri (iOS) versus Google Voice Search / Google Now
(Android 4.1) — run as a live matched-question battery, easy factual queries first, then harder ones.
The verdict is split by strength, and the analytically interesting bit is that he weights **source
transparency** as a real differentiator: Google shows you where every answer came from; Siri quietly
reads from Wolfram Alpha without saying so.

## Key claims
- [2012-07-01] **Verdict split by task, not a single winner**: "Google Now and Google Voice Search tend
  to be really really good with factual stuff — birthdays, weather, distances, sizes… thanks to Google's
  algorithms. Siri, which is a lot more conversational, can tell you jokes, can have conversations, can
  give you advice — Google search won't do that."
- [2012-07-01] **Source transparency treated as a differentiator** — the notable analytical criterion:
  "with Google Now and Google Voice Search you can actually see the source that's being used… if you get
  a card for an answer it'll give you a source, and if it gives you an image you'll also get the source
  for that image." By contrast, Siri "uses a lot of Wolfram Alpha in its results… but Siri won't read you
  off the Wolfram Alpha card." Provenance is scored as a feature.
- [2012-07-01] **A gracious tell — both fall back to Google**: "both of these competitors, if nothing
  else works, if you ask it a question it can't answer, it just defaults to a Google web search. So that
  says a lot about Google." Even Siri's fallback is Google — offered as a wry point rather than a dunk.
- [2012-07-01] **Reports his own likely behavior change**: "I personally think that Google Voice Search
  is really impressive, might actually start using it for more factual stuff." A dated, honest statement
  of intent rather than a detached verdict.
- [2012-07-01] **Re-frames the "Siri is a gimmick" reputation fairly**: "I know that Siri was considered a
  bit of a gimmick by a lot of people at first because it was very gimmicky-seeming… it had a lot of
  features that people would never use — but the factual stuff is very helpful." He credits the useful
  core while acknowledging the gimmick reputation.
- [2012-07-01] **Methodological transparency**: "you can see all the questions I'm asking in the link in
  the description" — publishing the test set so the comparison is reproducible/auditable.
- [2012-07-01] **Excludes a competitor honestly, with the reason**: "I was going to include Samsung's
  S Voice, but it was incredibly slow and not even worth trying and including in this comparison." Names
  what he left out and why — no silent omission.
