- [2009-05-06] **His setup**: he is running Windows Vista Home Premium 64-bit, which provides the Aero transparency and transparent toolbar he wants Safari to adopt.
- [2009-05-06] **The default problem**: out of the box, Safari 3 on Windows shows Apple's Mac OS X styling — "tiny little mac ma OS 10 buttons" and Apple's own toolbar — instead of the Aero theme.
- [2009-05-06] **The fix**: right-click Safari (shortcut or desktop icon) → Properties → Compatibility tab → check "Disable visual themes" → Apply → OK; from then on Safari opens with the Aero theme.
- [2009-05-06] **Why it works (his explanation)**: Apple applies its own visual theme to Safari's toolbar; Windows wants to apply the Aero toolbar to every application "unless it has a different sort of theme like a game or something," so disabling the app's visual theme lets Windows' Aero take over.
