# Tutorial: Firefox Addons (theme) (@mkbhd, 2009-02-01)

## Summary
An origin-era customization how-to (a non-HD tutorial): 15-year-old Marques installs a
**Firefox theme** ("Vista Arrow") that restyles Firefox to look like Internet Explorer,
then shows how to switch back and uninstall it. He explains version compatibility (check
Help → About Firefox; he's on **3.0.5**; incompatible themes appear greyed out) and notes
that uninstalling a theme leaves your other add-ons intact. A companion to his same-day
add-on install/uninstall tutorial.

## Key claims
- [2009-02-01] **How-to: install a Firefox theme** ("Vista Arrow," ~1.8 MB) via Add to
  Firefox → accept EULA → install → restart, then apply it under Tools → Add-ons → Themes.
- [2009-02-01] **Version-compatibility check:** Help → About Firefox shows your version
  (his is 3.0.5); themes for old versions are greyed out / flagged as incompatible.
- [2009-02-01] Uninstalling a theme and restarting returns Firefox to its previous look;
  "all your old add-ons remain, nothing's gone."

## Notable verbatim quotes (Marques — voice data)
> "it's going to make it look like Internet Explorer … it actually looks a lot like Internet Explorer, it's quite good, looks decent"

> "that's as easy as it takes to uninstall a Firefox add-on — bam, right back into the old Firefox theme"
