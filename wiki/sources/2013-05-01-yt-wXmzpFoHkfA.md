# Samsung Galaxy S4 Review! (@mkbhd, 2013-05-01)

## Summary
The full Galaxy S4 review, led by an original structural read — **the S4 is really a
downsized, reconstructed Note 2**, not an evolutionary S3. Awards it the **best all-around
smartphone camera** he'd used, but repeats his **function-over-gimmick** stance (he turned
the "smart" TouchWiz features off) and closes on the **is-it-worth-the-upgrade / buy-by-
your-values** framing, with the updates-as-value corollary.

## Key claims
- [2013-05-01] **The original read — "a downsized reconstructed Note 2"**: "a lot of people see
  the S4 as an evolutionary update to the S3… I'm seeing it as more of a downsized reconstructed
  Note II." Backs it with a physical teardown comparison (speaker/battery/SIM placement all
  match the Note 2, not the S3) — an observation most reviewers missed, evidence-based.
- [2013-05-01] **Best all-around smartphone camera he'd used**: "I'd say the best camera on any
  smartphone I've ever used — including the iPhone 5, Note 2, S3, HTC One, Lumia 920. This tops
  them all for me… the crown of best all-around camera." A dated, specific verdict.
- [2013-05-01] **Function-over-gimmick — turned the "smart" features off**: "Samsung stuffed
  this phone with software features… great for demos, great for showing off, but a lot won't be
  used by the average user. Smart Scroll, Smart Stay, Smart Pause… I ended up turning these off
  after a while — they got annoying and I just didn't need them." His demoable-≠-useful stance,
  lived.
- [2013-05-01] **Display verdict — pixels win, brightness is the trade-off**: the 1080p Super
  AMOLED "has a pentile subpixel matrix, but nobody can tell… razor-sharp, I love it, it's the
  number one reason to upgrade," but "no brighter than the S3, so it struggles in daylight vs
  the HTC One." Perception-over-spec (pentile doesn't matter) plus an honest trade-off.
- [2013-05-01] **The upgrade question + buy-by-your-values**: "is it worth your upgrade? If you
  have to have 1080p AMOLED, yes — this is the only way to go. If not, you might be better off
  waiting… the software features are around the corner in a firmware update for the S3 and Note
  II." The updates-as-value corollary and his don't-upgrade-unless-it-matches-your-priorities
  framing.
- [2013-05-01] **Even-handed on plastic — pros and cons**: "plastic is a pro (better signal, a
  lighter/thinner phone, fast GPS lock) but also cons (fingerprint magnet, feels cheap) — that
  decision comes down to what you prefer." Names the trade-off both ways.
- [2013-05-01] **Small-touch appreciation + a personal-taste pick**: credits the new metallic
  grip band ("a small touch a lot of reviewers look past… it really changes the feel"); "I'd go
  with the black version for the clean look and less fingerprints."
