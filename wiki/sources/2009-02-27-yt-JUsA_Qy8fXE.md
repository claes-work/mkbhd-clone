# HD Tutorial: Firefox 3.1 Beta 2 (@mkbhd, 2009-02-27)

## Summary
An install-and-first-look tutorial (Marques, age 15) for the **Firefox 3.1 beta 2** on Windows
Vista, a companion to his Safari 4 beta video. He covers the Mozilla download, the custom
install (not setting it as default yet), and warns that **many add-ons/themes are incompatible**
with the beta — he has to uninstall his Mac OS X and graphite themes. The standout is an early
**benchmark comparison**: he runs the **Acid3** web-standards test and reports **Safari 4 beta
scored 100/100 while Firefox 3.1 beta stops just short at 93**. Closes by dragging the new icon
into RocketDock. Reinforces the origin-era comparative-review reflex.

## Key claims
- [2009-02-27] **Acid3 test: Safari 4 beta = 100/100; Firefox 3.1 beta = 93.**
- [2009-02-27] Many Firefox add-ons and themes are **not compatible** with the 3.1 beta;
  he uninstalls his incompatible Mac OS X and graphite themes.
- [2009-02-27] Prior installed version was Firefox 3.0.6; the beta takes ~24-25 MB on disk and
  replaces the current install. Notes he's "heard a lot of good things about the speed."
