# HD Tutorial: Offline Gmail (@mkbhd, 2009-02-08)

## Summary
Origin-era how-to riding a then-current news cycle: Google had just enabled **offline
Gmail** as a Gmail Labs feature. Marques (age 15) shows enabling it under Settings → Labs,
and — for non-Chrome browsers — installing **Google Gears** from gears.google.com to let
web apps interact with the desktop. He notes Gears works with Firefox 1.5+ (he is on
**Firefox 3.0.6**), 32-bit Windows Vista, Internet Explorer 6+, and Safari. A snapshot of
the pre-cloud-native era when "offline web app" was a novel, install-a-plugin affair.

## Key claims
- [2009-02-08] **How-to:** enable offline Gmail via Settings → Labs; outside Chrome, it
  requires installing the **Google Gears** browser plugin.
- [2009-02-08] **Compatibility (dated):** Gears supported Firefox 1.5+, IE6+, Safari, and
  32-bit Windows Vista at the time; Chrome had the capability built in.
