# HD Tutorial: Start Menu Customization (@mkbhd, 2009-02-08)

## Summary
Origin-era two-part customization screencast. First, Marques (age 15) restores the classic
**Run** command to the Vista Start menu (Taskbar Properties → Start Menu → Customize →
check "Run command"), noting it works for both the classic and Vista Start menus and that
Vista's Start-search box already doubles as Run. Second, he shares his own personal setup:
he **reduced the recent-programs count from ~9-10 down to 8** to keep the Start menu
compact, and pins his most-used apps (browser, email client, Media Center, DVD suite, disk
defragmenter) to the top by dragging them above the separator. An early look at his taste
for a deliberately tidy, personalized workspace.

## Key claims
- [2009-02-08] **How-to:** re-enable the **Run** command box in the Vista Start menu via
  Taskbar Properties → Start Menu → Customize → "Run command"; reverse the same steps to
  remove it.
- [2009-02-08] **Personal setup (dated):** trimmed recent-programs display to **8** to
  shorten the Start menu, and pins frequently-used apps above the separator so they stay
  put regardless of usage.

## Notable verbatim quotes (Marques — voice data)
> "I decreased it to eight to shorten the size of my start menu so when I actually hit start it only goes about a little more than halfway up the screen"

> "it will always be there no matter how many other programs I open in this bottom eight it will be up here"
