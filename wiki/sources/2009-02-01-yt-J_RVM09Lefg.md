# HD Tutorial: Add the 'New Tab' button to Firefox (@mkbhd, 2009-02-01)

## Summary
A short origin-era **how-to**: 15-year-old Marques shows how to add a one-click **"New
Tab" button** to **Mozilla Firefox** (matching Internet Explorer's) via the
**New Tab Button (position)** add-on from addons.mozilla.org. He walks the install
(search add-ons → add to Firefox → restart), then tours the options for where the button
sits on the tab bar. His stated motivation is small-but-telling: he uses tabs heavily and
would "rather just hit one button — it's that much faster," an early expression of the
efficiency/quality-of-life framing that runs through his tech takes.

## Key claims
- [2009-02-01] **How-to: add a "New Tab" button to Firefox** using the "New Tab Button
  (position)" add-on (v0.6.0) from addons.mozilla.org; configurable position on the tab
  bar.
- [2009-02-01] **Efficiency framing:** prefers a single click over File → New Tab because
  "it's that much faster," an early quality-of-life/efficiency instinct.
