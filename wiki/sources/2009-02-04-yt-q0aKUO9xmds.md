# HD Tutorial: Virtual Memory (@mkbhd, 2009-02-04)

## Summary
Origin-era Vista performance screencast. Marques shows how to increase the Windows paging
file (virtual memory) via Computer → Properties → Advanced system settings → Performance →
Settings → Advanced → Virtual Memory → Change. His rule of thumb: set virtual memory to
**~1.5× installed RAM**. He notes his own rig — **4 GB RAM, a 7200 RPM hard drive** — and
sets 6 GB of virtual memory. He openly acknowledges others have covered this ("a lot of
people have already done videos on this, but i'm just going to do my own"), an early sign
of his stance that his production/clarity is the differentiator, not novelty of topic.

## Key claims
- [2009-02-04] **How-to:** raise Vista virtual memory via System Properties → Performance
  → Advanced → Virtual Memory → custom size (value entered in megabytes).
- [2009-02-04] **Rule of thumb (dated):** set virtual memory to approximately **1.5× your
  installed RAM** (he uses 6 GB for 4 GB of RAM).
- [2009-02-04] **His 2009 rig:** 4 GB RAM and a **7200 RPM** hard drive.
- [2009-02-04] **Creator stance (dated):** knowingly re-covers a well-trodden topic in
  his own way rather than avoiding it.
