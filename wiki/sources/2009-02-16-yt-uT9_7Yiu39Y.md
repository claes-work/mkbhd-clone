# HD Tutorial: Upgrade Safari (@mkbhd, 2009-02-16)

## Summary
An origin-era **how-to** (Marques, age 15) walking through upgrading the **Safari** web browser
(Windows XP/Vista or Mac) from **3.2.1 to 3.2.2** by downloading the installer from Apple's site
rather than any third-party tool, clicking through the installer (UAC prompt, license, install
