# HD Tutorial: Custom Desktop Icons (Request) (@mkbhd, 2009-05-14)

## Summary
A short origin-era Windows Vista customization screencast: how to replace a folder's icon
with a downloaded icon pack (source: iconspedia.com) via right-click → Properties →
Customize → Change Icon → Browse. Notable mostly as a data point in two established
origin threads: the **audience-request loop** (he titles it "(Request)" even though no
single viewer asked — the question had arrived repeatedly by YouTube message and through
his website) and the **macOS-in-Windows aesthetic** (the icon pack he uses is one he saved
as "Mac OS X icons"). Not an L3-candidate — routine tutorial.

## Key claims
- [2009-05-14] **"Request" in the title can mean aggregated demand, not one request**:
  "this one's not actually a requested video but I put a request in the title because this
  has actually been asked by a number of people" — via YouTube messaging or his website.
- [2009-05-14] **He already has a website contact channel** in addition to YouTube
  messaging ("or from contacting me through a website").
- [2009-05-14] **Method**: search "icon pack" → iconspedia.com (linked in the description)
  → download an `.ico` or `.png` (no install needed) → folder right-click → Properties →
  Customize tab → Change Icon → Browse.
- [2009-05-14] **Tradeoff he flags**: changing a folder icon means Windows "will no longer
  show a preview of the folder's contents" — a small but characteristic
  cost/benefit callout inside an otherwise mechanical how-to.
- [2009-05-14] **`.ico` vs `.png`**: either works; the `.ico` file is "a little bit
  smaller, if that makes a difference to you."
- [2009-05-14] **Built-in Vista icons rejected on taste grounds**: "there's a bunch of
  actually numerous built-in icons into Windows Vista but I don't really like any of
  these and I'd rather use my own that I downloaded."
- [2009-05-14] **The icon pack is a Mac OS X set on a Windows machine**: "I named it
  Mac OS 10 icons" — continues the documented origin-era macOS-look-on-Windows pattern
  (RocketDock, OS X Firefox themes, OS X cursors).
- [2009-05-14] **Link-in-description habit**: "I'll go ahead and throw that link in the
  description so you guys don't need to remember it."
- [2009-05-14] **Explicit request pipeline in the outro**: "if you need a request video
  done go ahead and subscribe and send me a message and I will try to get to you with a
  request video."
