# HD Tutorial: Google Chrome (out of beta) (@mkbhd, 2009-05-29)

## Summary
A ~90-second origin-era software video: how to install Google Chrome, and how to update an
existing install (wrench menu → "About Google Chrome" → Update now). Marques (age 15) frames
it as half tutorial, half **announcement** — Chrome has left beta. The substantive part is
the closing personal note: he says Chrome is "probably the fastest web browser out there
right now" and that he "might actually consider using this web browser now," having stayed
on Safari specifically to avoid running a beta browser. That is an early, dated statement of
a **stability-over-novelty default** — notable because he was simultaneously running the
Safari 4 *beta*, an inconsistency he names himself on camera.
