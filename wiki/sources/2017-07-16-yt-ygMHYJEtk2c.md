# Nest Cam IQ: Security Camera + 4k Sensor? (@mkbhd, 2017-07-16)

## Summary
Review of the Nest Cam IQ security camera — a good look at his growing smart-home footprint (he's a
multi-product Nest user) and a clean teach-the-mechanism on **using a 4K sensor for lossless zoom** rather
than recording 4K constantly. Verdict: better-in-every-way but only worth it if you don't already have a
Nest cam.

## Key claims (paraphrase, dated)
- [2017-07-16] **He's an invested Nest user (smart-home biography).** "I've been a Nest user for a couple
  months — cameras, thermostats, smoke detectors — so it's easy to keep it all in one app." A dated marker of
  his smart-home adoption and the value of an ecosystem.
- [2017-07-16] **Teach-the-mechanism: the 4K sensor is for lossless zoom, not 4K recording.** "It records in
  1080p at all times (recording 4K constantly would be insane), but because it's a 4K sensor you can zoom in
  up to 4× anywhere in the frame and stay full 1080p — basically lossless zoom (up to 12× on the live feed)."
  Explains what the headline spec actually buys.
- [2017-07-16] **Facial recognition fixes the false-alarm problem — but with a software gap.** "The old cam
  sent notifications for random sunlight changes or a blowing leaf ~99% of the time; the IQ uses facial
  recognition to only notify me about unknown faces/humans, and 'familiar faces' lets it name known people.
  But the software gap: **it won't let me turn off notifications for known faces and keep them for unknown
  ones** — a pretty obvious thing to add." Praises the smarts, names the missing option.
- [2017-07-16] **Design: unobtrusive-but-visible (a security-philosophy choice).** "Do you hide a security
  cam to secretly catch footage, or put it in plain view to discourage anything? I chose visible — pointed at
  the doorway. It's a clean, modern two-tone panda design (EVE-from-WALL-E look), USB-C powered." Values that
  it blends in while being deliberately visible.
- [2017-07-16] **Verdict: better-in-every-way, but upgrade-only-if-you-have-nothing.** "If you have a current
  Nest cam, you probably don't need to upgrade. But if you have no security cameras, I highly recommend
  getting *something* — this fits the premium peace-of-mind slot for people (including me) who value that. So
  I like them." A clear who-should-buy segmentation. (Notes best features tie to the paid Nest Aware sub.)
