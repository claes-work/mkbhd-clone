# Transformer Pad Infinity gets Jellybean! (@mkbhd, 2012-10-01)

## Summary
A short follow-up: the Asus Transformer Pad Infinity finally received Android 4.1 Jelly Bean, and he
checks whether it delivered the fix he'd **predicted** in his July review. It did — Project Butter
resolves the ICS hardware/software inconsistency — which closes his own conditional verdict. Also a
useful note on how the tablet UI differs by screen size (10" vs the Nexus 7's 7").

## Key claims
- [2012-10-01] **His July conditional verdict is confirmed** — the payoff: in the review
  ([[2012-07-23-yt-ddI-ZZwFclc]]) he'd said the Infinity would be "the best Android tablet money can buy
  *once it gets Jelly Bean*," grounding that in the Nexus 7 already running JB well. Here: "performance is
  extremely fast, it is all improved now… Project Butter is just that awesome. Pretty much brings the
  stability we've wanted… this is what Android was meant to be." The evidence-backed forecast lands.
- [2012-10-01] **Tablet UI is size-dependent** — a clarifying observation: "anything greater than the
  size of the Nexus 7 (a unique 7-inch form factor) still gets this tablet UI — you don't get the swipe-
  down notification bar, instead you get the regular horizontal layout with notifications in the
  bottom-right." Documents how Android 4.1 renders differently on 10" vs 7", tying back to the
  form-factor-matters thread.
- [2012-10-01] **Google Now on a 10" 1080p display uses the space**: "instead of just going to a search
  it brings you into the Google Now layout with two columns of cards… using all of the resolution of this
  1080p display as real estate." A UI-scales-with-the-screen note.
- [2012-10-01] **Stock-preference restated via a specific dislike**: "the one Asus thing I think is really
  ugly is the notification bar — I happen to think that looks really ugly, so I definitely don't enable
  it, I unchecked that." He turns off the skin option on sight, consistent with the near-stock preference.
- [2012-10-01] Confirms the update reached the whole Transformer line (Infinity, Prime, Pad 300).
