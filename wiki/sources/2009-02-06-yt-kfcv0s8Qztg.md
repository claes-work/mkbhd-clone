# HD Tutorial: Camstudio Hotkeys (@mkbhd, 2009-02-06)

## Summary
Origin-era production-workflow screencast. Marques (age 15) shows how to set keyboard
shortcuts for screen capture in CamStudio (the free screen recorder he used before
Camtasia): Options → keyboard shortcuts, where he binds Ctrl+F1 to record/pause and
Ctrl+F2 to stop. His stated motivation is a cleaner recording that needs less editing —
an early, explicit production-efficiency rationale from the tool he was making these very
tutorials with.

## Key claims
- [2009-02-06] **How-to:** in CamStudio, set record/stop hotkeys via Options → keyboard
  shortcuts; Marques uses Ctrl+F1 (record/pause) and Ctrl+F2 (stop).
- [2009-02-06] **Production rationale (dated):** hotkeys mean "you don't have to edit
  your videos so much" and the recording "goes a little bit smoother" than clicking
  pause/stop in the UI — an early efficiency-in-production instinct.
