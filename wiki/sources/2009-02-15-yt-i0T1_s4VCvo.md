# HD Tutorial: Eliminate Background Tasks (Request) (@mkbhd, 2009-02-15)

## Summary
A **request-driven** origin-era how-to (for a user transcribed **"cundiff13"**) on freeing
**RAM** by shutting down background applications in **Windows XP or Vista**. Marques (age 15)
opens **Task Manager → Processes**, sorts by memory, and points out background hogs — e.g. a
**Firefox preloader** eating RAM while the browser is closed, plus IM clients, minimize-to-tray
apps, and **Thunderbird**. He shows unloading them from the **system tray** (right-click →
unload/exit/close), then reopens Task Manager to prove the win: the top memory user is now just
the **Desktop Window Manager at ~46 MB**. Framed partly as a battery/performance tip (less RAM
pressure → fewer hard-drive hits).

## Key claims

- [2009-02-15] Rationale: fewer background tasks → **less RAM pressure → fewer hard-drive
