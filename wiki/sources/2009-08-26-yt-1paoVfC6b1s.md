# HD Tutorial: Custom Dock Icons (@mkbhd, 2009-08-26)

## Summary
Short **desktop-customization how-to**: 15-year-old Marques shows how to replace the
default icon on an ObjectDock/RocketDock entry with a custom PNG (finds a transparent-
background image via image search, saves it, then uses the dock's "Change Image" dialog
to swap it in). Demonstrated by fixing the ugly default Camtasia Studio icon with a
digital-camcorder PNG. Continues the running dock-customization thread on the channel.

## Key claims
- [2009-08-26] **How-to: swap a dock icon** on ObjectDock/RocketDock via right-click →
  Dock Entry Properties → Change Image, sourcing a transparent (PNG) image by search
  first.
- [2009-08-26] **PNG-for-transparency as a stated technical requirement** — an early,
  explicit awareness of file-format/transparency mechanics feeding into his
