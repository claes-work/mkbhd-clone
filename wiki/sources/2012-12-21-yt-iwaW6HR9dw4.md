# Hackintosh Pro Project! [Part 1] (@mkbhd, 2012-12-21)

## Summary
The pilot for a new multi-part build series: a **Hackintosh** — his own parts running OS X — built as
"a beast of a video-editing machine for the beginning of 2013." Two things make it a persona source:
he **confirms the macOS transition** ("I've been using a Mac Pro for the past couple of months for my
video production and I've gotten very used to OS X"), and he re-runs the **"true desktop" reasoning** to
justify a Hackintosh over every shipping Mac — while adding the decisive factor: **price** (a Hackintosh
is ~half a Mac Pro).

## Key claims
- [2012-12-21] **macOS transition confirmed and dated**: "a lot of you might not know, but I've been using
  a Mac Pro for the past couple of months for my video production, and I've gotten very used to the OS X
  environment, so I'm going to continue using OS X." Corroborates [[2012-09-21-yt-kRB1-APKUqs]] and
  [[2012-03-02-yt-G_H6mRHS9pc]] — the PC builder is now editing on a Mac.
- [2012-12-21] **The "true desktop" reasoning re-run to justify the Hackintosh** — the analytical spine:
  he rules out laptops ("not powerful enough for fluid video editing"), the retina MacBook Pros ("fans are
  loud"), the Mac Mini ("basically a glorified laptop"), and the iMac ("isn't very upgradeable except the
  RAM, once") — "so you only have the Mac Pro and the Hackintosh." The upgradeability doctrine from
  [[2012-10-23-yt-PPEpTasDz2s]] applied to his own build decision.
- [2012-12-21] **Why build instead of buy — two reasons stated**: (1) **Apple's delayed Mac Pro** — "I was
  using [the Mac Pro] hoping Apple would release a new one with a very powerful chip soon, but they just
  slacked on that, so we're taking our own initiative"; and (2) **price** — "an equivalently spec'd-out
  Hackintosh might cost half of what the Mac Pro costs." Price-to-value applied to his own gear.
- [2012-12-21] **Dual-boot Windows + OS X**, with the choice put to the audience: "I'm not sure if I want
  Windows 7 or Windows 8 because I'm so used to Windows 7… leave a comment and we'll pick." Audience-
  programmed, and a reminder he still runs Windows too.
- [2012-12-21] **Documenting the whole process as a series** — the project-series format: "I'll be
  documenting the selection process, the build process, the install process, the how-to… benchmarks,
  testing, gaming, and actual video editing." Transparency about the build as content.
- [2012-12-21] **Home for the holidays — a student marker**: "it's almost Christmas… I'm home for the
  holidays, as you can probably already tell." Dates a student living pattern (away at school, home for
  break).
- [2012-12-21] Cross-references the $600 PC giveaway ([[2012-12-13-yt-FFuePFTf9LQ]]) and teases "something
  special at the end of the series."
