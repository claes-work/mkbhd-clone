# HD Tutorial: Non-Ultimate Dreamscene (@mkbhd, 2009-02-02)

## Summary
Origin-era customization how-to: Marques shows how to get **Windows Vista Ultimate's
DreamScene** feature (a video playing as an animated desktop background) on *non-Ultimate*
editions of Vista, via a third-party download from **mydigitallife.info**. He installs it,
sets a video as the wallpaper through Personalize → Desktop Background, and argues the
non-Ultimate version is light on resources (under ~10 MB of RAM), keeping performance fine
while multitasking. Small tell: he sets **one of his own recently-uploaded YouTube videos**
as the background clip.

## Key claims
- [2009-02-02] **How-to:** search "non ultimate dreamscene download," grab the file from
  **mydigitallife.info**, extract and install to enable DreamScene on any Vista edition.
- [2009-02-02] **Resource claim:** the non-Ultimate DreamScene uses less RAM than the
  official Ultimate feature — "not even 10 megabytes," leaving performance fine while
  running a browser, email, Notepad, etc.
- [2009-02-02] **Reversible:** you can switch back to a static image any time via
  Personalize → Desktop Background.
