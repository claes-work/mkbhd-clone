# HD Tutorial: Get OS X Theme on Rocket Dock (@mkbhd, 2009-02-08)

## Summary
Origin-era customization screencast. Marques (age 15) walks through skinning **RocketDock**
(a third-party macOS-style dock for Windows) with a Mac OS X "Leopard" theme downloaded
from a skins site, extracting it into `Program Files/RocketDock/Skins`, then applying it
via Dock Settings → Style and repositioning the dock to the bottom of the screen with an
edge offset above the taskbar. One of a cluster of early videos in which he imports a
**macOS look into Windows Vista** — an early, documented pull toward Apple's aesthetic
years before he became a tech reviewer.
