# HD Tutorial: Install Windows Themes (Request) (@mkbhd, 2009-02-24)

## Summary
A by-request tutorial (Marques, age 15) on installing a third-party **Windows 7-look theme
onto Windows Vista**, sourced from **deviantArt** (compared to crystalxp.net). He models his
now-characteristic vetting habit — reading the comments before downloading to confirm the
file works — then walks the RapidShare free-tier download, WinRAR extraction, copying the
theme into `Windows\Resources\Themes`, the UAC allow prompt, running the theme installer, and
applying it via Personalize → Themes. He notes he doesn't actually apply it, only previews.
Made "in-depth" specifically because so many users requested it — an early instance of
audience-driven content.
