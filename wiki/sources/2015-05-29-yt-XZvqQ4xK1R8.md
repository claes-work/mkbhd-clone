# Top 5 Android Marshmallow Features! (@mkbhd, 2015-05-29)

## Summary
A Top-5 walkthrough of the Android M developer preview (Marshmallow) on his Nexus 6 — "a 5.2-style update
focused on polish." His #1 is **Google Now on Tap** (contextual info from any screen — "borderline
genius," if slightly creepy), grounded in his standing view that **Google Now has basically no real
competition** (Siri/Cortana aren't close). Also notable: **Doze** (idle-detection standby-battery, which he
loves), native fingerprint API + Android Pay, per-app permissions, Google Photos ("give iCloud a run for
its money"), and a **USB-C hint for the next Nexus.**

## Key claims (paraphrase, dated)
- [2015-05-29] **#1 Google Now on Tap.** Hold home on any screen → contextual cards about what's on-screen
  (a business, a movie in a tweet) — "some might say creepy, but it's super helpful and borderline genius."
  Grounded in: "Google Now already has basically zero competition — Siri and Cortana are pretty good voice
  assistants, but they don't get anywhere near its functionality, and Google keeps using its data to make it
  better."
- [2015-05-29] **#2 Doze.** Reads the sensors to detect an idle device and cuts background activity to
  extend standby — "I love any improvement to standby time. Hopefully this brings phones/tablets closer to
  the iPad, where I can leave it on a shelf a week and it still has battery."
- [2015-05-29] **#3 the little tweaks:** **per-app permissions** (revoke a specific app's mic/camera —
  "very nice security features"), word-by-word copy/paste, a settings light/dark theme, dual-band hotspot.
  (One dislike: the new vertical-scrolling app drawer is "uglier — hopefully they switch it back.")
- [2015-05-29] **#4 native fingerprint API + Android Pay** — a wide-open API to encourage OEMs and app
  makers (banking, in-app purchases) to build in fingerprint auth.
- [2015-05-29] **#5 volume controls "finally right"** — "it's been wrong for so long in stock Android…
  it's awesome to finally see it here" (silent/DND, separate media/ring/notification sliders).
- [2015-05-29] **Ecosystem notes:** **Google Photos** ("a free giant archiving/backup/search service —
  should give iCloud a run for its money"), Android Pay, and **USB-C support** — "a pretty good idea of what
  port to see in the next Nexus."
