# HD Tutorial: Dual Docks (Request) (@mkbhd, 2009-03-17)

## Summary
A **viewer-requested** screencast (Marques, age 15) showing how to run **two desktop docks at
once** on Windows — pairing his existing **RocketDock** (bottom of screen) with **ObjectDock**
(top of screen), so one dock holds applications and the other holds folders. He walks through
finding ObjectDock via a Google search, downloading the **freeware** version from
objectdock.com / the CNET download link (noting there is also a paid **"Plus" version for
$20**), and — since he already has it installed — skips the install as trivial. He then demos
ObjectDock's **Configure Dock** options: moving the dock to the top of the screen and
**reserving the edge of the screen** so maximized windows don't overlap it, which he likens to
the macOS dock behavior. Closes with his standard request-me-a-tutorial call to action.

## Key claims
- [2009-03-17] You can run multiple docks simultaneously on Windows by combining **RocketDock**
  with **ObjectDock**; he uses one for apps, one for folders.
- [2009-03-17] **ObjectDock** is freeware with a paid **Plus** tier (~$20); the free version is
  enough for a second dock.
- [2009-03-17] ObjectDock can **reserve the screen edge** (maximized windows won't cover it),
  which he compares to macOS dock behavior — an early instance of his recurring
  "make-Windows-feel-like-a-Mac" desktop-customization theme.
