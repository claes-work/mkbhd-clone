# HD Tutorial: Edit Web Pages with JavaScript (@mkbhd, 2009-06-05)

## Summary
A short, playful software tutorial: paste the `document.body.contentEditable`-style
JavaScript snippet into the address bar and any web page becomes locally editable. Two
things make this page worth more than the trick. First, an **unprompted biography fact**: he
demonstrates on "the website that I'm actually writing for right now" — the captions render
it "text support alert.com," almost certainly **TechSupportAlert.com** (he calls it "a
website full of free[ware]"), which would make him a **writer for an outside tech site at age
15**, alongside the YouTube channel (self-reported, not verified). Second, it corroborates
the **1,000-subscriber** mark: he edits his own channel page and jokes about changing "1,000
subscribers" to "1.3 million." He also mentions using the trick to briefly fool his parents
with a fake 330,000-view count before revealing it. ★ L3-candidate.
