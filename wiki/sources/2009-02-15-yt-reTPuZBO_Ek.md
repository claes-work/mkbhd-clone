# HD Tutorial: Create A YouTube Intro Video (Request) (@mkbhd, 2009-02-15)

## Summary
A **request-driven** origin-era tutorial in which Marques (age 15) documents **exactly how he
made his own channel intro** — the same intro shown in the companion upload
[[2009-02-14-yt-f4B3FMSFqpE]] (a music-only intro animation, not separately ingested). Workflow:
build a short slideshow at **animoto.com** (sign up → Create Video → 30-second short → upload
webcam/digital-camera photos → pick a royalty-free track from **Animoto's** library to avoid
copyright issues → render; Animoto emails the finished video). Then **export to YouTube**,
**download the file with a YouTube downloader**, **cut off the trailing Animoto branding in
Windows Movie Maker**, and set the result as the **HD intro prepended to every one of his
videos**. He explicitly frames this as the **"amateurish" free method** — not Flash, not highly
customizable, but it gets real results. He credits the request to a YouTube user (**"younome"**,
per caption) and reiterates his **do-any-request** content policy.
