# HD Tutorial: Thunderbird Addons (@mkbhd, 2009-02-01)

## Summary
An origin-era how-to for installing add-ons into **Mozilla Thunderbird**, the email
client: download the add-on file in a browser (he recommends Firefox), then in Thunderbird
go Tools → Add-ons → Install and point it at the downloaded file. He shows the add-ons he
personally runs — **Lightning** (integrated calendar + tasks, which he says he already
uses "quite a bit"), a minimize-to-tray add-on, and **ThunderBrowse** (adds in-client web
browsing). Confirms his real first name via the on-screen Windows user folder ("Marcus
downloads" = the "Marques" user).

## Key claims
- [2009-02-01] **How-to: add Thunderbird add-ons** — right-click → Save Link As to
  download the add-on file, then Thunderbird Tools → Add-ons → Install → open the file;
  restart once or twice and the add-on is integrated.
- [2009-02-01] **His own Thunderbird add-ons:** Lightning (calendar/tasks — already in
  active use), a minimize-to-tray add-on, and ThunderBrowse (in-client web browsing).
- [2009-02-01] Recommends Firefox as the browser for downloading the add-on file.
