# HD Tutorial: Download and Install Internet Explorer 8 (@mkbhd, 2009-04-04)

## Summary
An origin-era freeware install screencast (Marques, age 15) walking through upgrading from
**Internet Explorer 7 to IE8**, noting IE8 is now **completely out of beta**. He downloads it
via **Safari** (stressing the browser you download with "really doesn't matter"), selects
**Windows Vista** and English on the IE8 homepage, runs the ~**13.3 MB** installer, and after
letting it run offscreen returns to a "restart to install the latest Windows updates" prompt.
He frames the video as an **installation-only** video and explicitly a **video response to his
Safari 4 beta video**, promising fuller IE8 reviews later — part of the running browser cluster
(Safari/Firefox/IE) in his first-months tutorial series.

## Key claims
- [2009-04-04] **IE8** shipped final (out of beta); the installer is ~**13.3 MB** and, coming
  "from Microsoft to a Windows machine," downloads fast.
- [2009-04-04] The **browser used to download** an installer is irrelevant — he downloads IE8
  in Safari — a small early usability point.
- [2009-04-04] He runs on **Windows Vista**; IE8 finished with a "restart to install the latest
  Windows updates" step.
- [2009-04-04] Positions this as **installation-only**, a **response to his Safari 4 beta video**,
  with "more reviews about IE8 later to come."
