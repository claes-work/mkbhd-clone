# HD Tutorial: Revert to Classic Windows Start Menu (@mkbhd, 2009-02-10)

## Summary
Short origin-era how-to. Marques (age 15) shows how to switch **Windows Vista** to the
**Classic Start menu** (right-click taskbar → Properties → Start Menu tab → Classic),
notes it gives the older-Windows look and functionality, warns it's slow the first time it
loads and drops a few icons on the desktop, then shows how to revert. He frames reverting
to the modern Vista menu as preserving recently-used programs and customizations — and
pitches the classic option for anyone who dislikes the Vista menu or finds it slow to load.

## Key claims
- [2009-02-10] **How-to:** enable the **Classic Start menu** in Vista via taskbar
  Properties → Start Menu tab; reverting is the same path.
- [2009-02-10] The classic menu loads slowly the first time and adds desktop icons; the
  default Vista menu preserves recently-used programs and customizations.
